# EPC Dependency Planner

EPC工事の工程依存関係をGUIで編集する静的Webアプリです。

## 起動

`index.html` をブラウザで開くと動作します。ビルドや依存関係のインストールは不要です。

## 主な機能

- ネットワーク図でタスクをドラッグ配置
- 依存線モードでタスク間の FS / SS / FF / SF 関係を追加
- CPMによる計算工期、余裕日数、クリティカルタスク表示
- ガントチャートで計算日程を確認
- 表形式でタスク、期間、制約、先行タスクを一括編集
- JSONインポート / エクスポート
- ブラウザの localStorage に自動保存
