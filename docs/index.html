操作ガイド

迷ったらここだけ確認

基本は「タスクを作る」「依存をつなぐ」「必要なビューで確認する」の3つです。

基本の流れ タスク追加 → 詳細編集 → 依存接続 → ガント/表で確認 → JSON出力
1

タスクを追加・編集する

上部の「タスク追加」で工程を増やし、右の詳細で名称、期間、進捗、予定日を編集します。ネットワーク上のカードはドラッグで移動できます。

2

依存関係をつなぐ

「依存接続」を押してから、先行タスク、後続タスクの順にクリックします。依存種別、ラグ日数、線ラベルは上部の接続設定で指定します。

3

線を修正・削除する

ネットワーク上の依存線をクリックすると、右の詳細に接続元、接続先、依存種別、ラグが出ます。そこで修正や削除ができます。

4

見方を切り替える

ネットワークは関係整理、ガントは日程感、表は項目確認と一括編集に向いています。検索と領域フィルターは各ビューに共通で効きます。

5

安全に試す

間違えたら「編集戻す」「編集進む」で戻せます。共有や保管をしたい場合は「JSON出力」、受け取った工程は「JSON取込」を使います。

ネットワーク タスクの前後関係を見ながら、カードと線を直接操作する画面です。
ガント 依存関係から計算された開始位置と期間を横棒で確認する画面です。
表 コード、状態、期間、進捗、予定完了などを一覧で確認・編集する画面です。
